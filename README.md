### Installation
1. Clone this repository
```console
$ git clone git@github.com:CarlSmoky/portfolio.git
```

2. Install packages
```console
$ cd portfolio
$ npm install
```

3. Start React APP
```console
$ npm run start
```

4. Run the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
